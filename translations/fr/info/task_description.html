Vous recevez une expression contenant des nombres, parenthèses et opérateurs. Pour cette mission, seules les parenthèses importent. Elles se présentent sous 3 formes: "{}" "()" or "[]". Elles sont utilisées pour déterminer la portée ou restreindre des expressions. Si une parenthèse est ouverte, elle doit être fermée avec une parenthèse de fermeture du même type. La portée d'une parenthèse ne doit pas interférer avec une parenthèse d'un autre type. Dans cette mission,vous devez décider de la cohérence de l'expression suivant les parenthèses. Ne vous occupez pas des opérateurs et opérandes.

Entrée: Une expression (chaîne de caractères - str) avec différents types de parenthèses (unicode).

Résultat: Une valeur de cohérence de l'expression en valeur booléenne (True ou False).

Exemple:

checkio("((5+3)*2+1)") == True
checkio("{[(3+1)+2]+}") == True
checkio("(3+{1-1)}") == False
checkio("[1+1]+(2*2)-{3/3}") == True
checkio("(({[(((1)-2)+3)-3]/3}-3)") == False
checkio("2+3") == True

Utilisation: Quand vous écrivez du code ou des expressions complexes dans un package mathématique, vous pouvez avoir un mal de tête intense quand il y a des erreurs de parenthèses. Ce concept peut être utile pour votre IDE personnel.

Précondition:
Il y a seulement des parenthèses ("{}" "()" or "[]"), des nombres ou des opérateurs ("+" "-" "*" "/").
