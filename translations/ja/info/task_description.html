あなたは数字、括弧と演算子を持つ式が与えられています このタスクでは括弧だけを対象にします。 括弧は3種類あります:"{}" "()" または "[]"。 括弧は範囲(scope)を決めたり、いくつかの式を制限したりするために使われます。 もし括弧が開いているなら、同じタイプの閉括弧で閉じなければいけません。 括弧の範囲は他の括弧と交わってはいけません。 このタスクであなたは括弧に基づいて式が正しいかどうかを決めなければいけません。 演算子とオペランドは気にしなくて構いません。 (訳注: 式が文字列として与えられるので、括弧の対応付けが正しいかどうかを判定してください。括弧以外の数字や記号は無視して構いません。)

入力: 異なるタイプの括弧をもつ式、文字列(unicode)

出力: 式が正しいかどうか、ブール値(TrueまたはFalse)

例:

{{init_code_tmpl}}

どのように使われるか: 数学パッケージでコードや複雑な式を書くとき、多かったり少なかったりする括弧に頭をかかえることでしょう。 このコンセプトはあなた自身のIDEに役に立ちます。

事前条件:
There are only brackets ("{}" "()" or "[]"), digits or operators ("+" "-" "*" "/").
