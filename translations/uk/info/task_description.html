Вам дані деякі вирази, що містять цифри, дужки та оператори. Для цього завдання нам будуть потрібні тільки дужки. Дужки бувають 3 типів: "{}" "()" or "[]". Дужки використовуються для визначення меж або для обмеження деяких виразів. Якщо дужка відкрита, то це означає, що повинна бути друга дужка такого ж типу, яка її закриває. Дужки різних типів не повинні перетинатися. У цьому завданні ви повинні вирішити, корегувати вираз чи ні. Рішення повинне базуватися на правильності розташування дужок у виразі. Не переймайтесь щодо операторів та операндів.

Вхідні дані: Вираз, з різними типами дужок у вигляді рядка (unicode).

Вихідні дані: Вердикт правильності розташування дужок у заданому виразі у вигляді (True або False).
